# 职宠小窝 APP 项目发现记录

## 代码结构分析

### 项目架构

```
app/lib/
├── core/                    # 核心层
│   ├── constants/           # 常量定义
│   ├── errors/              # 异常和失败类
│   └── theme/               # 主题系统 (颜色、字体、间距、阴影)
├── data/                    # 数据层
│   ├── datasources/         # 数据源 (本地/远程)
│   │   └── local/           # SQLite 数据源
│   ├── models/              # 数据模型
│   └── repositories/        # 仓库实现
├── features/                # 功能模块
│   ├── columns/             # 专栏模块
│   ├── confide/             # 倾诉室模块
│   ├── jobs/                # 职位模块
│   ├── notifications/       # 通知模块
│   ├── park/                # 公园模块
│   ├── profile/             # 个人中心模块
│   └── stats/               # 统计模块
├── routes/                  # 路由配置
├── services/                # 服务层
│   ├── confide_service.dart
│   ├── intent_engine.dart   # 意图识别引擎
│   └── pet_responder.dart   # 宠物响应逻辑
└── shared/                  # 共享组件
    └── widgets/             # 通用UI组件
```

### 架构评价

**优点:**
- 使用 Feature-based 目录结构，模块边界清晰
- 主题系统封装完善，支持亮色/暗色模式
- 意图识别引擎设计合理，支持扩展

**待改进:** 部分已解决

**已解决:**
- ✅ 数据层 (Repository Pattern) - 已实现 SQLite + DAO
- ✅ 错误处理机制 - 已添加异常类和失败类
- ✅ 日志系统 - 已添加 Logger

**待改进:**
- 依赖注入 (可考虑使用 get_it)
- 深色模式完善

### 意图识别引擎 (IntentEngine)


**特点:**
- 规则-based 实现，轻量高效
- 支持否定语境识别 (如 "没投简历" → slacking)
- 易于扩展新的意图类型

### 主题系统

**文件:** `core/theme/`

- `app_colors.dart` - 颜色定义
- `app_typography.dart` - 字体样式
- `app_spacing.dart` - 间距规范
- `app_shadows.dart` - 阴影效果
- `app_theme.dart` - 主题组合

### 动画效果

- `AnimatedStar` - 星星闪烁动画
